# Proof Bundle

This directory holds proof artifacts generated by governed workflows.

## Structure

- `core/` — Replay-critical hot proof artifacts (core.json, summary.md, etc.)
- `extended/` — Large or secondary evidence referenced by digest

## Hot vs Extended

The split ensures that replay-critical truth stays on the hot path while
large auxiliary evidence lives in referenced cold storage. Replay must
succeed from core artifacts alone.

## Status

These directories are placeholders created by `keyhole init vertical`.
They will be populated by later governed run and proof workflows.
