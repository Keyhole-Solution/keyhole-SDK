# Public Runtime Bridge Surface

This surface is subordinate to CE-V5-S40-07. The test runtime is a public
boundary consumer and does not claim platform topology authority.

## Local-Only Mode

Local-Only mode is reproducible for development and smoke tests. It is not constitutional proof and does not imply upstream audit evidence.

## Governed Mode

Governed Mode requires `KEYHOLE_MCP_URL` and `KEYHOLE_MCP_TOKEN`. In governed
mode the runtime may attach candidate isolation evidence and public governance
receipt fields returned through the MCP boundary.
