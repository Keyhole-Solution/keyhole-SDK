Trademarks

This document describes the trademark and brand usage guidelines for Keyhole and related marks used in this repository.

The Keyhole name and associated marks are the property of Keyhole Solution Foundation.

Keyhole Marks

The following names and marks are trademarks or service marks of Keyhole Solution Foundation:

Keyhole

Keyhole Platform

Keyhole Developer Kit

Keyhole Test Runtime

Additional product names, logos, or marks associated with the Keyhole ecosystem may also be protected.

All rights are reserved except as described below.

Acceptable Use

You may use the Keyhole name and trademarks in the following ways:

Descriptive reference

You may refer to Keyhole in documentation, articles, or code when describing integrations or compatibility.

Examples:

compatible with the Keyhole Test Runtime
built using the Keyhole Developer Kit
integration with the Keyhole platform
Open-source contributions

Contributors to this repository may reference Keyhole marks within:

source code

documentation

examples

commit messages

issue discussions

when describing functionality related to this project.

Tutorials and educational material

Developers may create tutorials, guides, or educational material referencing the Keyhole Developer Kit or the Keyhole Test Runtime.

Restricted Use

You may not use Keyhole trademarks in ways that could cause confusion about ownership, endorsement, or affiliation.

This includes:

Product names

You may not name a product in a way that suggests it is an official Keyhole product.

Examples of not allowed uses:

Keyhole Cloud
Keyhole Enterprise Server
Official Keyhole Platform
Misleading branding

You may not:

use Keyhole logos or marks in a way that implies endorsement

represent your software as the official Keyhole platform

present a fork of this project as the official Keyhole Developer Kit

Domain names and company names

You may not register domain names or companies using the Keyhole mark in a way that creates confusion about official ownership.

Examples:

keyhole-platform.com
keyholecloud.io
Logos and Visual Branding

Logos and visual branding assets may be subject to additional usage restrictions.

Unless explicitly provided for public use, Keyhole logos should not be used in external branding or marketing material without permission.

Open Source License vs Trademarks

The open source license covering the code in this repository does not grant trademark rights.

Even if code is freely licensed, the Keyhole name and associated marks remain protected.

Reporting Misuse

If you believe Keyhole trademarks are being used in a misleading or unauthorized way, please report the issue to:

legal@keyholesolution.com

Include:

the location where the mark is being used

a description of the issue

any relevant screenshots or links

Updates

These guidelines may be updated as the Keyhole ecosystem evolves.

Please refer to the latest version of this document in the repository for current trademark policies.
